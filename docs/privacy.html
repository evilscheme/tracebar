Privacy Policy

Effective March 5, 2026

No Data Collection

TraceBar does not collect, store, or transmit any personal data. The app contains no analytics, telemetry, crash reporting, or tracking of any kind.

Local Only

All network diagnostic data — including IP addresses, hostnames, and latency measurements — exists only in your Mac's memory while the app is running. This data is discarded when TraceBar closes. Nothing is written to disk or sent to any server.

User Preferences

Your settings (target host, color scheme, probe intervals, etc.) are stored locally using macOS UserDefaults. These preferences never leave your device.

Network Access

TraceBar sends ICMP echo request packets to the target host you specify, for the sole purpose of traceroute monitoring. No other network communication occurs. The app does not contact any first-party or third-party servers.

No Accounts

TraceBar requires no sign-up, login, or user identification of any kind.

Third-Party Code

TraceBar contains no third-party libraries, SDKs, or frameworks. It is built entirely with Apple system frameworks.

Contact

If you have questions about this privacy policy, contact tracebar@evilscheme.org.
