## Version Information

**Promotional Text:**
> Real-time network path monitoring — see every hop between you and any host, right from your menubar.

**Description:**
> TraceBar puts a live traceroute in your macOS menubar. It continuously monitors the network path to any host, showing every hop's latency, packet loss, and route changes in real time.
>
> Features:
> - Continuous traceroute monitoring with adaptive probe intervals
> - Menubar sparkline showing destination latency at a glance
> - Per-hop heatmap and sparkline visualizations
> - 13 color schemes for latency visualization
> - Configurable probe intervals, latency thresholds, and history windows
> - Automatic DNS resolution for hop hostnames
> - Launch at Login support
>
> Built for network engineers, developers, gamers, and anyone who wants visibility into their network path. TraceBar requires no root access or admin privileges — it uses unprivileged ICMP sockets that work inside the macOS App Sandbox.
>
> Your privacy matters: TraceBar collects no data, includes no analytics or tracking, and makes no network connections beyond the traceroute probes you configure. All diagnostic data stays on your Mac and is discarded when the app closes.
